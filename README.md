# GitHub API Experiments




This is an experimental project to understand how GitHub APIs work.

## Setup

This project uses the GitHub CLI (`gh`) for authentication and API interactions.

## Goals

- Learn GitHub REST API endpoints
- Understand GitHub GraphQL API
- Experiment with repository management
- Explore GitHub webhooks and events

## Getting Started

1. Ensure you're authenticated with GitHub CLI:
   ```bash
   gh auth status
   ```

2. Use the GitHub CLI to interact with the API:
   ```bash
   gh api <endpoint>
   ```

## Examples

- `examples/basic_api_example.py` - Basic REST API usage
- `examples/graphql_example.py` - GraphQL API examples
